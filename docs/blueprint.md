# **App Name**: REHANZA CRM

## Core Features:

- Dashboard UI: Responsive dashboard layout with a fixed sidebar and main content area. Includes top summary stat cards and a clean, modern UI.
- Inventory Management: Manage product inventory, including stock levels and product details.
- Pricing and Settlement: Manage pricing and settlement details for different marketplaces.
- Order Management: Track and manage orders from multiple channels like Meesho, Flipkart, and Amazon.
- Analytics Dashboard: Display analytics and profit tracking information to monitor performance.
- Low Stock Alerting Tool: AI-powered tool which detects low stock items.

## Style Guidelines:

- Primary color: Deep purple (#6366F1) for a professional and trustworthy feel.
- Background color: Light gray (#F9FAFB), nearly white, for the main content area.
- Accent color: Indigo (#4F46E5), analogous to purple but slightly more muted, for interactive elements and highlights.
- Body and headline font: 'Inter', a sans-serif, providing a clean and modern look for both headlines and body text.
- Use simple, consistent icons for menu items and dashboard elements.
- Fixed left sidebar (260px width) with gradient from purple-800 to indigo-900. Main content area in white with rounded-2xl cards and soft shadow-xl. Consistent spacing with p-6 gap-6.
- Subtle transitions and animations for user interactions (e.g., menu toggles, card hover effects).
